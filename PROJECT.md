# English Flow

## Mission

English Flow 是一个几乎无需操作的英语输入工具。

它不是 AI 聊天软件。
它不是背单词软件。
它不是 Anki。

它的目标只有一个：

> **尽可能降低学习操作成本，提高持续英语输入量。**

---

## Target User

主要供项目作者自己使用。

特点：

* 考研英语
* 六级约460分
* 不喜欢机械背单词
* 喜欢自动播放
* 喜欢边听边看
* AI负责解释，本项目负责输入

---

## Core Philosophy

减少点击。

减少人为操作。

增加英语输入。

让学习像刷短视频一样自然进行。

本项目优先保证：

1. 简单
2. 稳定
3. 自动
4. 长期可维护

---

## MVP

打开网页后即可开始学习。

流程：

单词

↓

自动朗读

↓

显示音标

↓

显示中文

↓

停顿

↓

自动进入下一词

整个流程尽量无需用户点击。

---

## 开发路线

Milestone 1（基础版本）

- 自动播放
- 自动朗读
- 自动切换
- 播放/暂停
- 速度调节

Milestone 2（学习功能）

- 单词数据
- 学习统计
- 今日学习汇总
- 历史学习记录

Milestone 3（复习功能）

- 记忆曲线
- 自动提醒
- 今日待复习

Milestone 4（句子模式）

- 输入句子
- 自动朗读
- 循环播放

---

## Out of Scope

本项目不负责：

* AI解释
* AI造句
* AI聊天
* AI词根分析
* AI作文
* AI翻译

以上能力全部交给 ChatGPT 等 AI 工具。

---

## Technical Stack

* HTML
* CSS
* Vanilla JavaScript
* LocalStorage

无需后端。

移动端优先。

简单优先。

稳定优先。

长期维护。

---

## Design Principle

AI负责：

* 编程
* 调试
* 解释

网页负责：

* 自动播放
* 自动朗读
* 学习记录
* 复习提醒

不集成AI聊天。

不调用在线大模型。

保持纯HTML + CSS + Vanilla JavaScript即可运行。

---

## Development Principle

* 一次只完成一个TODO。
* 不主动开发后续功能。
* 不为了一个功能重构整个项目。
* 优先修改已有代码，而不是重新生成全部代码。
* 完成一个TODO后更新TODO.md和CHANGELOG.md，并停止等待确认。

## 长期规划

未来支持：

- 多词库切换（考研/CET4/CET6/自定义）
- 每日学习目标设置
- 循环播放/播完暂停
- 标熟（Mastered）
- 记忆曲线复习
- 今日待复习列表
- 学习范围选择：
    - 全部单词
    - 未掌握（默认）
    - 今日待复习
    - 已标熟（浏览）
